# Gamepad to Keyboard

## Install

    apt install libxdo-dev 
    apt install libudev-dev 

## Use PS4 controller on Linux

http://ros-developer.com/2017/12/14/ps4-controller-bluetooth-ubuntu/

    sudo pip install ds4drv
    sudo ds4drv
